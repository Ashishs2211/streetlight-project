🎓 B.Tech Final Year Project

Computer Vision-Based Detection of
Streetlight Outages in Urban Areas

A complete deep learning pipeline using YOLOv8 to automatically detect non-functional streetlights from images, video, and live camera feeds.

YOLOv8 PyTorch OpenCV Smart City Public Safety Real-Time Detection
01 — Installation
terminal 
# 1. Create & activate virtual environment
python -m venv venv
source venv/bin/activate       # Linux / Mac

# 2. Install all dependencies
pip install -r requirements.txt

05 — Dataset Preparation

📥 Option A — Download from Roboflow

Search "streetlight detection" on universe.roboflow.com, export in YOLOv8 format, and place files in data/.

📷 Option B — Collect Your Own

Photograph streetlights at night (ON and OFF). Annotate with LabelImg using YOLO format. Aim for 500+ images total.

annotation — YOLO label format (.txt) 
# Each line: class_id  x_center  y_center  width  height  (all normalized 0–1)

0  0.512  0.423  0.045  0.120    ← functional   (light ON)
1  0.231  0.387  0.042  0.115    ← non_functional (light OFF)
0  0.789  0.401  0.048  0.118    ← functional
split_dataset.py 
# Place raw images in data/raw/images/ and labels in data/raw/labels/
python scripts/split_dataset.py

# Custom split ratios
python scripts/split_dataset.py --train 0.7 --val 0.2 --test 0.1
06 — Training
train.py — command examples 
# Basic training (50 epochs, batch 16)
python train.py

# Custom settings
python train.py --epochs 100 --batch 32 --model yolov8s.pt

# GPU training
python train.py --epochs 100 --batch 32 --device 0

# Model sizes: n (nano) → s → m → l → x (largest)
# For CPU: use yolov8n.pt   |   For GPU: use yolov8s.pt or larger
07 — Inference
detect_image.py + detect_video.py 
── Image detection ──────────────────────────────────────
python detect_image.py --source data/images/test/
python detect_image.py --source street1.jpg --show

── Real-time video / webcam ─────────────────────────────
python detect_video.py --source 0               # webcam
python detect_video.py --source night_road.mp4    # video file
python detect_video.py --source rtsp://ip/stream  # CCTV stream

── Evaluation ───────────────────────────────────────────
python evaluate.py --weights models/best.pt
